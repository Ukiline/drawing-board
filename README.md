## 简介
fabric + vue + socket-io 实现的在线直播画板

## 快速启动
```shell
npm install

npm run start
```
访问 http://localhost:8080

## 主要功能

* 画板翻页
* 修改画笔粗细
* 修改画笔颜色
* 绘制几何图形
* 添加文字
* 操作图片
* 上传背景
* 橡皮擦 (单选、框选)
* 清空画布
* 撤销、上一步下一步
* 复制粘贴
